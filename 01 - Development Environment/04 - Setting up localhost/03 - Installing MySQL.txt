
Downloading MySQL
-----------------

Go to: http://www.mysql.com/downloads/mysql/
Download the latest version that applies to your system. Login in order to download.


Installing MySQL
----------------

Install the .pkg file, and then drag the .prefPane file to the System Preferences application to install it, install it for all users.


Setting up the root user
------------------------

Start the MySQL server.
$ cd /usr/local/mysql/bin
$ ls -la
$ ./mysql -u root # To check if the root password is setup
$ exit
$ ./mysqladmin -u root password 'rootpwd'
