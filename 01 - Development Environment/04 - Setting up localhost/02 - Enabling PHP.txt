Stop the Apache server:
$ sudo apachectl graceful-stop

Go to the configuration file:
$ sudo nano /etc/apache2/httpd.conf

Control+w, type "php" press enter, uncomment the line deactivating PHP.
Control+x, y.

$ cd /private/etc
$ ls php*
$ sudo cp php.ini.default php.ini
$ sudo nano ./php.ini

Control+w, type "display_errors" press enter, do this twice.
Change the value from "Off" to "On".

You can turn Apache on now.
