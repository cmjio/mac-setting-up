- [Asepsis](http://asepsis.binaryage.com/)
  - After installing Asepsis you need to run the following commands:
```commandline
$ asepsisctl enable
$ asepsisctl install_wrapper
$ asepsisctl make_dscage
$ asepsisctl install_daemon
$ asepsisctl install_updated
```
